# Attendify - Face Recognition Attendance System

Attendify is an AI-powered Attendance Management System that uses Face Recognition to automate attendance marking. This system eliminates manual attendance and improves accuracy and efficiency.

## Features

- Face detection using OpenCV
- Face recognition using trained datasets
- Automatic attendance marking
- User-friendly GUI built with Tkinter
- Secure data storage using MySQL
- Student registration and login system
- Real-time face recognition

## Technologies Used

- Python
- OpenCV
- Tkinter
- MySQL
- NumPy

## How it Works

1. Register student details
2. Capture face images
3. Train the model
4. Recognize faces in real-time
5. Mark attendance automatically
